# 라즈베리파이 5 전용 패키지. requirements.txt 와 함께 설치한다:
#   pip install -r requirements.txt -r requirements-rpi.txt
# (RPi 5에서는 RPi.GPIO 대신 gpiozero + lgpio 백엔드 사용)
gpiozero
lgpio
gTTS              # 한국어 TTS (Linux의 pyttsx3/espeak는 한국어 미지원 → 챌린지 발화용)

# USB 마이크/스피커(각성 챌린지 음성)용 시스템 패키지 — pip 설치 전에 필요:
#   sudo apt install -y portaudio19-dev flac mpg123
#   (portaudio19-dev: PyAudio 빌드 / flac: STT 인코딩, aarch64는 내장 바이너리 없음 / mpg123: gTTS mp3 재생)

#   `sudo apt install -y python3-picamera2` 후 시스템 인터프리터를 사용한다.
